## 🚀 Key Highlights & Tech Stack

This project is built using a lightweight, performant, and zero-dependency frontend architecture for fast load speeds and modular rendering:

*   **Core:** React 19, TypeScript, JavaScript (ES6+), HTML5
*   **Styling & Design:** Vanilla CSS3 (Custom Glassmorphic Panels, Glowing Gradients, Floating Animations)
*   **Build Tooling:** Vite (Lightning-fast HMR and bundling)
*   **Features:** Custom scroll-spy navigation, custom typing text cursor, fluid transitions, and responsive mobile-first grid.

## 🛠️ Local Development & Setup

To run this portfolio website locally on your machine, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/ritik-hedau18/my-portfolio.git
    cd my-portfolio
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    *Open your browser and navigate to `http://localhost:5173` (or the port specified in terminal).*

4.  **Build for Production:**
    ```bash
    npm run build
    ```
